Админ-кабинет
=============

Админ-кабинет доступен только пользователям с ролью **ADMIN**. Открывается из шапки приложения (кнопка «Кабинет» или аналогичная). Внутри — вкладки: личные настройки, почта, доски, пользователи, архив.

.. image:: images/placeholder.png
   :alt: Вкладки админ-кабинета
   :align: center
   :width: 90%

Вкладка «Личное» (Personal)
---------------------------

На этой вкладке администратор может редактировать свой профиль так же, как в обычном модальном окне профиля: имя, доска по умолчанию, аватар, уведомления по email.

.. image:: images/placeholder.png
   :alt: Личные настройки в кабинете
   :align: center
   :width: 80%

Вкладка «Почта» (SMTP)
----------------------

Настройки SMTP для отправки писем: сброс пароля, уведомления участникам карточек. Задаются хост, порт, безопасное соединение, учётные данные, адрес отправителя. Есть кнопка «Проверить подключение» для тестовой отправки.

.. image:: images/placeholder.png
   :alt: Настройки SMTP в кабинете
   :align: center
   :width: 80%

Вкладка «Доски» (Boards)
------------------------

Список всех досок: создание, редактирование названия и описания, удаление. Для выбранной доски — управление колонками: добавление, переименование, изменение порядка (например, перетаскиванием), удаление, архивирование колонки (создаётся архив).

.. image:: images/placeholder.png
   :alt: Управление досками и колонками
   :align: center
   :width: 90%

Вкладка «Пользователи» (Users)
-----------------------------

Список пользователей: создание нового пользователя (email, имя, пароль, роль, **требование 2FA** по умолчанию включено), редактирование (почта, роль, **вкл/выкл требования 2FA**), сброс пароля, удаление. Для пользователя можно настроить доступ к доскам: выбор досок, к которым он привязан (PUT /api/auth/users/:id/boards).

.. image:: images/placeholder.png
   :alt: Управление пользователями и доступом к доскам
   :align: center
   :width: 90%

Вкладка «Архив» (Archive)
-------------------------

Список созданных архивов (архивы колонок или карточек). Для каждого архива доступны: скачивание файла архива, восстановление данных из архива в доску, удаление записи об архиве.

.. image:: images/placeholder.png
   :alt: Список архивов и действия
   :align: center
   :width: 90%

Итог по API: пользователи (GET/POST/PATCH/DELETE, сброс пароля, доступ к доскам), настройки почты (GET/PUT, тест), доски и колонки, архив (GET, download, restore, delete) — см. :doc:`api`.
