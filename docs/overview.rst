Обзор приложения
================

Назначение
----------

**Kanban** — веб-приложение для управления задачами в формате досок с колонками и карточками. Поддерживаются несколько досок, участники карточек, комментарии, вложения, двухфакторная аутентификация (2FA), сброс пароля по почте и админ-функции (пользователи, доски, колонки, архивы, настройки почты).

Стек технологий
---------------

* **Backend:** Node.js, Express, Prisma ORM, PostgreSQL, сессии (connect-pg-simple), TOTP для 2FA.
* **Frontend:** React, Vite, Tailwind CSS, @dnd-kit для перетаскивания карточек и колонок.
* **Развёртывание:** Docker Compose, опционально HTTPS; скрипты резервного копирования и восстановления БД.

Роли пользователей
------------------

* **ADMIN** — полный доступ: управление пользователями, досками, колонками, настройками SMTP, архивами, доступ пользователей к доскам.
* **MEMBER** — работа с назначенными досками: просмотр и редактирование карточек, комментарии, вложения, участники; смена пароля и 2FA в профиле.

Основные сущности
-----------------

* **Доска (Board)** — контейнер колонок и карточек; у пользователя может быть доска по умолчанию.
* **Колонка (BoardColumn)** — столбец на доске с заголовком и порядком; карточки принадлежат колонке.
* **Карточка (Card)** — задача: описание, детали, исполнитель, срок, важность, пауза; комментарии, вложения, участники.
* **Пользователь (User)** — учётная запись с email, именем, аватаром, ролью, настройкой уведомлений по почте и 2FA.
